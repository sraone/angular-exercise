# Installation

Clone this repo and open terminal or command line. Navigate project folder  

#### Install npm packages
```sh
$ npm install
```

#### Install Bower packages
```sh
$ Bower install
```
# Run

#### Build project
```sh
$ grunt build
```

#### Run project
```sh
$ grunt serve
```

#### Test project
```sh
$ grunt coverage
```
